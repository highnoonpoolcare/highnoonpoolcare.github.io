# High Noon Pool Care Website

Professional pool maintenance website for High Noon Pool Care, built with [Astro](https://astro.build/) + [Tailwind CSS](https://tailwindcss.com/) using the [AstroWind](https://github.com/onwidget/astrowind) template.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Static output goes to `dist/`.
